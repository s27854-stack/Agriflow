🔧 ESP32 Setup Guide

  1. Install ESP32Servo library
    Tools → Manage Libraries → "ESP32Servo"
  2. Moisture Sensor → GPIO 34, Servo → GPIO 18
  3. Set server URL:
    serverUrl = "http://your-pc-ip:3000/api/sensor"
  4. Threshold & duration adjustable from dashboard!
  5. Flash to ESP32, Serial Monitor at 115200
